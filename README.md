## 🧐 Como usar




#### Para instalar

  > Clonar o repositório
   
        $ git clone https://github.com/Rafael-Yokoyama/Projetos-Trilha-Abinbev-React.git
        $ cd projeto_food


#### Para rodar 
  > Instalar dependências
    
       $ yarn
       
       
  > Iniciar o Projeto 
  
       $  yarn start
